### 🚀 CUDA 细胞自动机 (生命游戏) 运行说明

感谢下载！本程序通过 NVIDIA CUDA 技术实现，将原本由 CPU 负担的复杂计算完全交由 GPU 并行处理。

#### 💻 运行环境
*   **显卡需求**：必须使用 **NVIDIA 显卡**（支持 CUDA）。
*   **驱动要求**：请确保显卡驱动已更新至较新版本。
*   **系统**：Windows 10 / 11 (64-bit)。

#### 🕹️ 操作快捷键 (参考)
*   **[左键点击/拖拽]**：绘制细胞（注入能量）。
*   **[右键点击/拖拽]**：擦除细胞。
*   **[空格键]**：暂停 / 继续演化。
*   **[滚轮]**：缩放视图。

#### ⚠️ 注意事项（必看）
1.  **杀毒软件误报**：由于程序未经过昂贵的商业数字签名，Windows Defender 或 360 可能会拦截。请点击“仍要运行”或添加白名单，本程序绿色安全，无任何后台行为。
2.  **缺少 DLL 文件**：如果提示缺少 `cudart64_xx.dll`，请确保安装了 NVIDIA 显卡驱动。
3.  **非 NVIDIA 显卡**：很抱歉，AMD 或 Intel 显卡用户暂时无法运行本 CUDA 版本。

#### 🌟 技术亮点
*   **并行演化**：数百万细胞实时模拟。
*   **动态热力图**：展示细胞消亡的视觉拖尾。
*   **计算不可约性**：你正在观察一个无法被公式预言，只能被模拟的真实宇宙。
